Website ini merupakan website sekolah SMKN 2 Surabaya yang dibuat menggunakan HTML dan CSS vanilla sebagai proyek website pertama saya. Website ini dirancang sebagai website profil sekolah yang menampilkan berbagai informasi penting, seperti halaman utama (Home), profil sekolah, visi dan misi, program keahlian (Vocational Program), kontak sekolah, serta tautan menuju website resmi SMKN 2 Surabaya.
